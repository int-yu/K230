# K230 本地 RTSP 网页查看器

浏览器不能直接播放 RTSP。本工具仅在 `127.0.0.1:8765` 提供网页，并调用本机 FFmpeg 将 K230 的 H.264 RTSP 转为 MJPEG。

## 使用

1. K230 终端出现 `Wi-Fi RTSP started: rtsp://<IP>:8554/test`。
2. 双击 `start_viewer.bat`。
3. 在网页输入终端打印的地址，只点击一次“开始播放”。

查看器全局只维护一个 FFmpeg/K230 上游连接。重复点击、按 Enter 或打开多个标签页不会创建多个 RTSP 上游。10 秒没有首帧会停止 FFmpeg，并在网页显示错误。

网页仅绑定 `127.0.0.1`，局域网内其他设备无法访问。
